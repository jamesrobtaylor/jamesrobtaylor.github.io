Wireshark Cheat Sheet

Basic Operations

Action Description
Ctrl + E Start/Stop capturing packets
Ctrl + K Restart capturing packets
Ctrl + S Save captured packets
Ctrl + O Open a capture file
Ctrl + Q Quit Wireshark

Display Filters

Filter Description
ip.addr == [address] Filter by IP address
tcp.port == [port] Filter by TCP port
udp.port == [port] Filter by UDP port
http Filter HTTP traffic
dns Filter DNS traffic
icmp Filter ICMP traffic
eth.addr == [address] Filter by Ethernet address
frame contains "[text]" Filter by frame content

Capture Filters

Filter Description
host [address] Capture traffic to/from a specific host
net [network] Capture traffic on a specific network
port [port] Capture traffic on a specific port
tcp Capture TCP traffic
udp Capture UDP traffic
icmp Capture ICMP traffic
ether host [address] Capture traffic to/from a specific Ethernet address

Statistics

Command Description
Statistics > Summary Show summary of the capture
Statistics > Protocol Hierarchy Show protocol hierarchy
Statistics > Conversations Show conversations between endpoints
Statistics > Endpoints Show endpoint statistics
Statistics > IO Graphs Show input/output graphs

Follow Streams

Command Description
Analyze > Follow > TCP Stream Follow TCP stream
Analyze > Follow > UDP Stream Follow UDP stream
Analyze > Follow > HTTP Stream Follow HTTP stream

Miscellaneous

Command Description
Ctrl + F Find a packet
Ctrl + G Go to a specific packet
Ctrl + ↑ Move to previous packet
Ctrl + ↓ Move to next packet
Ctrl + B Set time reference to selected packet
Ctrl + T Set a time reference
