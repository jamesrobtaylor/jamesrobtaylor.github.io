Basic Commands

Command Description
msfconsole Start Metasploit console
search <keyword> Search for exploits, payloads, auxiliary modules
info <module> Display detailed information about a module
use <module> Use a specific module
set <option> <value> Set options for a module
show options Show options for a module
show payloads Show available payloads
show encoders Show available encoders
show nops Show available NOPS
show post Show available post-exploitation modules
set payload <payload> Set the payload to be used with the exploit
exploit Run the exploit

Networking Commands

Command Description
ifconfig Show network interfaces
route Show routing table
route add <subnet> <netmask> <gateway> Add a route
route remove <subnet> <netmask> <gateway> Remove a route

Database Commands

Command Description
db_connect <user>:<pass>@<host>/<database> Connect to the database
workspace -a <workspace_name> Create a new workspace
workspace <workspace_name> Switch between workspaces
hosts List hosts in the database
services List services in the database
vulns List vulnerabilities in the database
notes List notes in the database

Meterpreter Commands

Command Description
background Background the current session
sessions -l List all background sessions
sessions -i <session_id> Interact with a background session
upload <local_path> <remote_path> Upload a file to the victim machine
download <remote_path> <local_path> Download a file from the victim machine
execute -f <command> Run a shell command on the victim machine
screenshot Take a screenshot of the victim machine's desktop

Auxiliary Modules

Command Description
use auxiliary/scanner/<scanner_module> Use an auxiliary scanner
set RHOSTS <target_ip_range> Set RHOSTS for scanners
run Run an auxiliary module

Post-Exploitation Modules

Command Description
use post/<post_exploit_module> Use a post-exploitation module
set SESSION <session_id> Set SESSION for post modules
run Run a post module
