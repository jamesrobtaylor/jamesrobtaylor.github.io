Basic Commands

Command Description
pwd Print working directory
ls List directory contents
cd [directory] Change directory
cp [source] [destination] Copy files or directories
mv [source] [destination] Move or rename files or directories
rm [file] Remove files
touch [file] Create a new empty file
mkdir [directory] Create new directory
rmdir [directory] Remove empty directory

File Viewing and Editing

Command Description
cat [file] Concatenate and display file content
less [file] View file content page by page
more [file] View file content with scrolling
head [file] View the first 10 lines of a file
tail [file] View the last 10 lines of a file
nano [file] Edit files with Nano editor
vi [file] Edit files with Vi editor

File Permissions

Command Description
chmod [mode] [file] Change file permissions
chown [owner][:group] [file] Change file owner and group

Searching and Finding

Command Description
find [directory] -name [filename] Find files by name
grep [pattern] [file] Search for a pattern within files
locate [name] Find files by name using a database
which [command] Show the path of a command

System Information

Command Description
uname -a Show all system information
df -h Show disk usage in human-readable format
du -sh [directory] Show directory size
top Display real-time system processes
free -h Show free and used memory
uptime Show how long the system has been running
who Show who is logged on
dmesg Display kernel message buffer

Networking

Command Description
ip a Show IP addresses and network interfaces
ping [host] Send ICMP ECHO_REQUEST to network hosts
netstat -tuln Show listening ports
ss -tuln Show listening ports (alternative to netstat)
curl [url] Transfer data from or to a server
wget [url] Download files from the web
scp [source] [user@host:destination] Secure copy over SSH
ssh [user@host] Connect to a remote host via SSH

Package Management

Command Description
apt update Update package lists (Debian/Ubuntu)
apt upgrade Upgrade all packages (Debian/Ubuntu)
apt install [package] Install a package (Debian/Ubuntu)
yum update Update package lists (RHEL/CentOS)
yum install [package] Install a package (RHEL/CentOS)
dnf update Update package lists (Fedora)
dnf install [package] Install a package (Fedora)

Disk and File System Management

Command Description
mount [device] [mountpoint] Mount a file system
umount [mountpoint] Unmount a file system
fdisk -l List disk partitions
mkfs.ext4 [device] Format a partition with ext4
fsck [device] Check and repair a file system

Scripting and Automation

Command Description
sh [script] Run a shell script
bash [script] Run a bash script
crontab -e Edit user crontab
at [time] Schedule a job at a specific time

Environment Variables

Command Description
echo $VARIABLE Print value of a variable
export VARIABLE=value Set an environment variable
env List all environment variables

User and Group Management

Command Description
useradd [user] Add a new user
passwd [user] Change user password
userdel [user] Delete a user
groupadd [group] Add a new group
usermod -aG [group] [user] Add user to group

Archive and Compression

Command Description
tar -cvf [archive.tar] [directory] Create a tar archive
tar -xvf [archive.tar] Extract a tar archive
gzip [file] Compress a file with gzip
gunzip [file.gz] Decompress a gzip file
zip [archive.zip] [file] Create a zip archive
unzip [archive.zip] Extract a zip archive

Miscellaneous

Command Description
alias [alias_name]='[command]' Create a command alias
history Show command history
!! Run the last command again
man [command] Show the manual for a command
echo [text] Print text to the terminal
date Show current date and time
cal Show calendar
