Hacking Tool Cheat Sheet

1. Information Gathering

Tool Description Example
Nmap Network discovery and security auditing. nmap -sP 192.168.1.0/24
Wireshark Network protocol analyzer. Open Wireshark GUI and start capturing on desired interface.
Recon-ng Web-based reconnaissance framework. recon-ng

2. Vulnerability Scanning

Tool Description Example
Nessus Vulnerability scanner. Use the Nessus web interface to scan network ranges.
OpenVAS Open-source vulnerability scanner. openvas-setup
Nikto Web server scanner. nikto -h http://example.com

3. Exploitation

Tool Description Example
Metasploit Penetration testing framework. msfconsole
BeEF (Browser Exploitation Framework) Exploit web browsers. ./beef
SQLmap Automated SQL injection tool. sqlmap -u "http://example.com/vulnerable.php?id=1"

4. Password Cracking

Tool Description Example
John the Ripper Password cracker. john --wordlist=password.lst hashes.txt
Hydra Brute force login cracker. hydra -l admin -P password.lst ftp://192.168.1.100
Hashcat Advanced password recovery. hashcat -m 0 -a 0 -o cracked.txt hashes.txt wordlist.txt

5. Wireless Attacks

Tool Description Example
Aircrack-ng Wi-Fi security auditing. aircrack-ng -w wordlist.txt -b 00:11:22:33:44:55 capture.cap
Reaver WPS PIN brute force tool. reaver -i mon0 -b 00:11:22:33:44:55
Kismet Wireless network detector. kismet

6. Post Exploitation

Tool Description Example
Empire Post-exploitation framework. ./empire
Mimikatz Windows post-exploitation tool. mimikatz

7. Social Engineering

Tool Description Example
Social-Engineer Toolkit (SET) Social engineering attacks. setoolkit
Maltego Open-source intelligence and forensics. Open Maltego GUI.

8. Web Application Testing

Tool Description Example
OWASP ZAP (Zed Attack Proxy) Web application security scanner. zap.sh
Burp Suite Web vulnerability scanner. Open Burp Suite GUI.

9. Miscellaneous Tools

Tool Description Example
Netcat Network debugging and exploration. nc -lvp 4444
Proxychains Redirect network connections through proxy servers. proxychains nmap -sT 192.168.1.0/24
Tor Anonymity online. tor
