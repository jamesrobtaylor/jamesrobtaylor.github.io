Nmap Cheat Sheet

Basic Scanning

Command Description
nmap [target] Basic scan of the target
nmap -v [target] Verbose scan of the target
nmap -A [target] Advanced scan with OS and version detection
nmap -T4 [target] Faster scan with timing template

Host Discovery

Command Description
nmap -sP [network] Ping scan to discover live hosts
nmap -sn [network] Ping scan without port scan
nmap -Pn [target] Scan without ping
nmap -PS [target] TCP SYN discovery on specified ports
nmap -PU [target] UDP discovery on specified ports

Port Scanning

Command Description
nmap -p [port] [target] Scan specific port
nmap -p [port-range] [target] Scan range of ports
nmap -F [target] Fast scan of top 100 ports
nmap -r [target] Scan ports consecutively
nmap --top-ports [number] [target] Scan top [number] ports

Service and OS Detection

Command Description
nmap -sV [target] Version detection
nmap -O [target] OS detection
nmap -A [target] Advanced scan with OS and version detection
nmap --script [script] [target] Run specific Nmap script
nmap -sC [target] Run default scripts

Output Options

Command Description
nmap -oN [file] [target] Normal output to file
nmap -oX [file] [target] XML output to file
nmap -oG [file] [target] Grepable output to file
nmap -oA [basename] [target] Output in all formats

Miscellaneous

Command Description
nmap -v [target] Increase verbosity
nmap -d [target] Increase debugging level
nmap --open [target] Show only open ports
nmap --reason [target] Show reason for each port state
nmap --packet-trace [target] Show all packets sent and received
nmap --iflist Show host interfaces and routes
